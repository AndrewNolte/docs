\subsection*{Ssh to Jetson}

how to find the ip address

run {\ttfamily ifconfig}

find the adreess that coresponds to the jetson ie enx000ec6da785e

run 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo arp-scan --localnet --interface=enx000ec6da785e}
\end{DoxyCode}


then ssh. Use -\/X for video forwarding 
\begin{DoxyCode}{0}
\DoxyCodeLine{ssh -X ubuntu@10.42.0.112}
\end{DoxyCode}
